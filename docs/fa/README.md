## ترتیب پیشنهادی مطالعه

**پروژه جدید روی لاراول ۱۳ به بالا**

معرفی → نیازمندی‌ها → نصب → شروع سریع → حالت‌های ذخیره‌سازی

**پروژه موجود در حال ارتقا از ۲.x یا ۳.۱**

نیازمندی‌ها → راهنمای ارتقا → مرجع دستورات

**فقط به شهرها نیاز دارید**

شروع سریع (با `--target=cities`) → حالت‌های ذخیره‌سازی
